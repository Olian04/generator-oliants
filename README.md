# generator-ts-quickstart
A yeoman generator for both Node and Web applications using Typescript

# All
* Generate `package.json` using `generator-npm-init`.
* Dev deps:
    * Typescript
    * Mocha
    * Chai
* @types for all deps

# Node
* Deps:
    * Lodash
* Dev deps:
    * Ts-node
    * Typedoc

# Web
* Dev deps: (everything is bundles, so theres no ordinary deps)
    * Lodash
    * React
    * Laravel-mix
